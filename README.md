# Voor Mevrouw Meray 🍉

Kleine one-page grap om Mevrouw Meray te laten flexen. Eén `index.html`, geen dependencies.

## Online zetten met GitHub Pages

1. Maak op GitHub een nieuwe (public) repository, bijv. `semra`
2. In deze map:
   ```bash
   git init
   git add .
   git commit -m "voor semra"
   git branch -M main
   git remote add origin https://github.com/<jouw-naam>/semra.git
   git push -u origin main
   ```
3. Op GitHub: **Settings → Pages → Branch: main / root → Save**
4. Na ±1 minuut staat hij op `https://<jouw-naam>.github.io/semra/`

Tip: `noindex` staat aan, dus zoekmachines slaan hem over.
